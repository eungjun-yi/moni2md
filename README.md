Transcode moniwiki text to markdown text.

Usage
-----

From a file

    moni2md MoniwikiText

From a stdin

    moni2md < MoniwikiText

Notice
------

Currently this supports only partial moniwiki syntax. Please open an issue or
send a pull request if you want more features.
